About the Project

This project is an integration of machine learning concept with the web development concept. A dataset about cars price is selected then preprocessed and used for training a ML model. This trained model is attached with this webapp. When user give the input then ML model will predict the price. Here we use Django as a web framework and Lasso Regression as machine learning model.

About the Dataset

This dataset is taken from the kaggle repository. This dataset contains information about used cars.This data can be used for a lot of purposes such as price prediction to exemplify the use of lasso regression in Machine Learning.
The columns in the given dataset are as follows:

  • name - Name of the car
  • year - Year of car when it was bought
  • selling_price - Price at which the car is being sold
  • km_driven - Number of Kilometres the car is driven
  • fuel - Fuel type of car
  • seller_type - Tells if a Seller is Individual or a Dealer
  • transmission - Gear transmission of the car
  • Owner - Number of previous owners of the car.
